üéØ The Problem

Managing multiple debts is overwhelming. Between credit cards, student loans, and other financial obligations, it's hard to know which debt to prioritize and how much to pay each month. That's why I built DebtFree Tracker.

üí° The Solution

A web application that provides:

  • Smart Financial Planning: Automatic calculation of available balance after savings
  • AI-Powered Recommendations: Personalized payment strategies using Google Gemini AI
  • Visual Dashboard: Clear overview of income, debts, and savings

üí™ Challenges & Solutions

Challenge 1: Securing API Keys

Problem: Exposing API keys in frontend code is a security risk.

Solution: Used Netlify Functions as a serverless proxy to keep API keys secure on the server side.

Challenge 2: Data Persistence Without Backend

Problem: Need to save user data without setting up a database.

Solution: Used localStorage with JSON serialization for client-side data persistence.

Challenge 3: Budget Validation

Problem: Users might add debts that exceed their budget.

Solution: Added real-time validation with helpful warnings when minimum payments exceed available income.

üìö What I Learned

  • Async/Await for API Calls: Handling asynchronous operations cleanly

  • Event Delegation: Efficiently handling dynamic elements
  • Serverless Functions: Keeping API keys secure with Netlify Functions
